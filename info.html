Info

This page describes how to use our bot and how it was made.

Messages

Our bot is based upon Telegram, so all your messages are secured by its protocol. See Telegram security and privacy policy for more information.

Our bot (in its application code) has access to all messages, that are sent to it. This is, obviously, required for bot to function properly. These messages are never stored on our servers, they only determine bot's next reply once. These messages are not processed in any other kind of way.

Infrastructure

Our bot consists of three major components: telegram bot, uber api client and web site (not this one). All three parts are hosted in a secure Microsoft's Azure cloud, see more on Azure security. All the communication with outer world (telegram, uber, site visitors) uses secure protocol: HTTPS. Local communication is secured by the Azure cloud itself.

Uber

Our bot uses official Uber API to communicate with Uber. Our bot does not and will never ask to type your passwords for any third party apps, sites, etc (including Uber and Telegram). When authorizing, the password is typed on the Uber official site. Our bot will never request rides or do any other communication with Uber API, unless expicitly requested by the end user.

To see details, check Uber docs, which we used as a primary guide.
